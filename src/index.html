Seleção de instrução

Instruções de uso:

  • Não é permitido o uso de espaços vazios antes, dentro ou depois da entrada.
  • Toda const deve ser escrita como CONST_{seuValor}.
  • Variável temp deve ser escrita como TEMP_{seuValor}.
